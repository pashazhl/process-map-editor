# process-map-editor

Редактор карты процессов с табличным и диаграммным представлением.

## Что внутри

- режим «Диаграмма» для просмотра и редактирования процессов;
- режим «Таблица» для массовой правки процессов и подпроцессов;
- режим «Системы» для редактирования системной карты и перетаскивания узлов;
- добавление и удаление процессов, подпроцессов, системных узлов и связей;
- светлая и темная темы;
- сохранение в `localStorage` браузера;
- импорт и экспорт JSON.

## Запуск

Откройте `index.html` в браузере. Сервер и сборка не нужны.

Для локальной проверки через HTTP:

```bash
python3 -m http.server 5174
```

Затем открыть `http://localhost:5174/`.
